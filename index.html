Governance · Process maturity

Process Maturity Framework

One common governance language across all frameworks. Every capability moves through four levels — Documented, Operationalized, Governed, then Optimized. Tick the evidence as each is met; the level is awarded automatically once all criteria for that level (and the ones before it) are checked. Governed is the target — a capability at Governed is 100% mature; Optimized is the bonus stage for continuous improvement.

This is the evidence register — the whether, and how we know: SOPs, success criteria, and owners behind each level. For the when — deadlines and SLA per gate — see Delivery Timeline. Same four-rung ladder, two lenses.
